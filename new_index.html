Featured Artifact: TRAVLR

TRAVLR is a full‑stack travel booking platform originally built in my Full‑Stack Development course. For this capstone, I enhanced it across three categories:

Software Design & Engineering

Implemented a modular encryption utility for sensitive data, added a user account/settings API, and refactored for maintainability.

Algorithms & Data Structures

Streamlined authentication with constant‑time hashing, consolidated validations, and optimized data flows.

Databases

Extended the MongoDB schema to include user settings, added CRUD operations with validation, and ensured proper indexing.

Explore the full narrative and see both the original and enhanced code on the TRAVLR page.
